# Datasets

RT-MLIDS is evaluated on two publicly available benchmark datasets. Due to size constraints, datasets are not included in this repository. Download them from the official sources below.

---

## NSL-KDD

An improved version of the KDD Cup 1999 dataset. Removes duplicate records and balances the test set for more reliable evaluation.

- **Source:** [Canadian Institute for Cybersecurity](https://www.unb.ca/cic/datasets/nsl.html)
- **Training records:** 125,973
- **Test records:** 22,544
- **Classes:** Normal, DoS, Probe, R2L, U2R

**Setup:**
```bash
mkdir -p data/NSL_KDD
# Place KDDTrain+.txt and KDDTest+.txt in data/NSL_KDD/
```

---

## CIC-IDS-2018

A contemporary dataset generated by the Canadian Institute for Cybersecurity with realistic network traffic.

- **Source:** [Canadian Institute for Cybersecurity](https://www.unb.ca/cic/datasets/ids-2018.html)
- **Records:** 16M+
- **Attack categories:** 14 (DDoS, DoS, Brute Force, SQL Injection, Infiltration, Bot, and more)

**Setup:**
```bash
mkdir -p data/CIC_IDS_2018
# Place CSV files in data/CIC_IDS_2018/
```
